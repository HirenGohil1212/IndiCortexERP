# **App Name**: ApexERP

## Core Features:

- Sales Management: Manage sales orders with 'Net Available' stock calculation, integrating inventory and production data.
- Purchase Management: Streamline purchasing with automated barcode tracking, invoice upload and integrated quality control (IQC) processes.
- Production Planning: Automate production with Bill of Materials (BOM), Routecards, and real-time shop floor data collection.
- Financial Accounting: Full financial suite with GST compliance and double-entry accounting to maintain accurate financial records.
- Trigger Engine: Centralized trigger system that sends automated WhatsApp notifications (via self-hosted gateway) and email updates based on defined rules.
- Dual Environment (Sandbox/Production): Provision of sandbox and production to test notifications, file uploads, barcode scanning safely before deployment
- Unified Dashboard with Role-Based Access: Provide comprehensive UI with all functionality like Inquiry entry to Balance sheet Generation and Role-based Access Control. Also every action needs to handled within single screen which can hosted AWS or local server.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for a professional, trustworthy feel.
- Background color: Light gray (#F0F4F9), a slightly desaturated shade of the primary color, offering a clean and modern backdrop.
- Accent color: Vibrant purple (#A737FF), analogous to the primary but with distinct brightness and saturation for clear visual cues.
- Body and headline font: 'Inter', a sans-serif font known for its legibility and modern appearance, ideal for both headings and body text.
- Use consistent, clear icons that match the modular structure of the ERP system.
- Employ a responsive, grid-based layout to ensure compatibility across devices, as this ERP must be mobile-friendly.
- Incorporate subtle transitions to guide users smoothly between different modules and actions within the dashboard.
